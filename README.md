# GitHub 测试项目

这是一个通过 OpenClaw 自动化创建的 Python 测试项目。

## 功能

- 打印 Hello World
- 显示当前日期、时间和星期

## 使用方法

```bash
python3 main.py
```

```
claude-gh-test/

```
